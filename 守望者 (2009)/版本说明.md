守望者 (2009) 有三个版本：


- 院线版 (Theatrical Cut) 162 分钟 (2小时42分钟)
    ```
    最早上映的版本;
    为了迎合院线删减了很多内容, 现在已经很难找到;
    ```

- 导剪版 (Director’s Cut) 186 分钟 (3小时6分钟)
    ```
    最佳版本, 也是导演本人最满意的版本;
    DVD中附带了一个30分钟的番外动画《黑货船传奇》;
    ```

- 终极版 (Ultimate Cut) 215 分钟（3小时35分钟）
    ```
    这是目前网上最流行, 最容易找到的版本;
    
    其实就是把动画《黑货船传奇》分几段插入导剪版中;
    增加了几段报刊亭的场景,用于衔接黑货船动画片段;
    增加了一句罗夏的台词;

    实际上插入的片段与电影主线关系不大, 还破坏了原片的叙事节奏, 初次观看强烈不推荐;
    ```

---

由于终极版的流行, 最适合观看的导演剪辑版资源反而变得难找.所以我决定基于终极版字幕, 调整时间轴和删减部分内容, 制作出最好的导剪版资源.

基于 @goodliuwu 的字幕 https://subhd.tv/a/L53zZp 做修改, 调整了时间轴, 并优化了部分翻译. 

熟肉会发布在阿里云盘: https://www.alipan.com/s/GJ5CHHy93mG

 (阿里云盘的好处是不用会员下载速度也还行, 至少下得动; 不会像百度/迅雷那样几十kb/s)

---

FFmpeg 压制命令, 上下加黑边并加字幕:
```bash
ffmpeg -i "Watchmen.2009.Directors.Cut.1080p.BluRay.DDP5.1.10bit.H.265-iVy.mkv" -vf "pad=1920:1080:0:140:black,subtitles='导剪版字幕.ass'" -c:v h264_qsv -q:v 18 -c:a copy "Watchmen.mp4"
```
